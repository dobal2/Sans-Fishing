> **고무게 물고기는 극히 드물다.** 70kg 이상 확률은 합산 0.25% 미만.

### 가치 계산 공식

```
valueRatio = (weight - minWeight) / (maxWeight - minWeight)
finalValue = floor(baseValue × (1.2 + valueRatio^0.8 × 7.0))
```

#### 무게별 가치 배율 (공식 기반)

| 무게 구간 | valueRatio | 배율 |
|---------|-----------|------|
| 최솟값 (minW) | 0.00 | ×1.20 |
| ~10kg | ~0.09 | ×1.75 |
| ~30kg | ~0.29 | ×2.96 |
| ~50kg | ~0.49 | ×4.07 |
| ~70kg | ~0.69 | ×5.13 |
| ~90kg | ~0.89 | ×6.16 |
| 최댓값 (100kg) | 1.00 | **×8.20** |

> **같은 물고기라도 100kg vs 1kg은 가치 차이가 약 6.8배**  
> Small Carp 최솟값: ~173코인 / 최댓값: ~1,181코인

### 낚싯대 성능 이상 분석

- **RainbowRod vs UnknownRod**: RainbowRod(50만)의 럭 보너스(+120%)가 UnknownRod(100만)보다 낮음. 500,000 → 1,000,000 투자 대비 럭 +50%p 증가 — 가성비 합리적.
- **RainbowRod 럭 역전 버그**: CloverRod(+130%) > RainbowRod(+120%). 더 비싼 RainbowRod가 럭에서 뒤짐. **밸런스 이슈**.
- **VoidRod 가격 장벽**: 3,355만 코인. ElectricRod 대비 약 15배 비쌈. 럭은 2배 수준(+300% vs +600%). **극단적 프리미엄**.
- **Power×2 게임패스**: RodPower를 2배로 만들어 VoidRod 효율을 사실상 두 배로 올림 — 과금 선호.

## 4. 미니게임 난이도 밸런스

### 희귀도별 미니게임 설정

| 희귀도 | 목표 진행도 | 히트존 크기 | 슬라이더 속도 | 초당 감소량 |
|--------|-----------|-----------|------------|---------|
| Common | 100 | 20~25% | 1.5~1.8s | 1/s |
| Uncommon | 120 | 18~22% | 1.5~1.8s | 1.5/s |
| Rare | 140 | 16~20% | 1.5~1.8s | 2/s |
| Epic | 160 | 14~18% | 1.5~1.8s | 3/s |
| Legendary | 180 | 12~16% | 1.5~1.8s | 4/s |
| Mystic | 200 | 10~14% | 1.5~1.8s | 5.5/s |
| Fantasy | 250 | 8~12% | 1.5~1.8s | 7/s |
| Unknown | 300 | 6~10% | 1.5~1.8s | 9/s |

### 진행도 게인/손실

| 이벤트 | 진행도 변화 |
|--------|----------|
| 강타 (Strong Zone) | +20 × progressMultiplier |
| 일반 히트 (Normal Zone) | +10 × progressMultiplier |
| 미스 | -15 (고정) |
| 패시브 감소 | -rarity drain rate / 초 |

### RodPower가 미니게임에 미치는 영향

```
targetProgress = baseProgress / rodPower  (최소 50)
hitZoneSize    = baseHitSize × (1 + (rodPower - 1) × 0.3)  (최대 0.30~0.35)
```

> VoidRod(Power 3.0) 기준:  
> Unknown 목표진행도: 300 / 3.0 = **100** (Common 수준으로 하락)  
> 히트존: 0.06 × (1 + 2×0.3) = **0.096** (6% → 9.6%)

### 슬라이더 속도 고정 문제

모든 희귀도에서 슬라이더 속도(1.5~1.8s)가 동일 — 난이도 차별화가 히트존 크기와 드레인 속도에만 의존. Fantasy/Unknown에서 빠른 슬라이더를 추가하면 더 다이나믹한 난이도 곡선 구현 가능.

## 5. 확률 시스템 종합 분석

### 럭 적용 우선순위 (누적 적용)

```
최종 확률 = 기본 확률 × (1 + 낚싯대 럭 보너스) × (럭 게임패스 보정) × (버블 버프 보정)
```

### 낚싯대 + 럭패스 + 버블 버프 조합 시 Unknown 확률

| 조합 | Unknown 확률 |
|------|------------|
| FishingRod, 패스 없음 | 0.10% |
| FishingRod + 럭패스 | 0.20% |
| VoidRod, 패스 없음 | 0.10% × (1+6.0) = 0.70% |
| VoidRod + 럭패스 | ~1.4% |
| VoidRod + 럭패스 + 버블버프 | ~3.5% |

> **Unknown 물고기 평균 시도**: VoidRod + 풀버프 기준 약 28~30회당 1회. 기본 FishingRod로는 1000회당 1회.

### 버블 버프 효과 (희귀도별 증감)

## 8. 현재 밸런스 문제점

### 🔴 심각

1. **RainbowRod 럭 역전**: RainbowRod(500,000, +120%) < CloverRod(250,000, +130%)  
   → RainbowRod 럭을 +150~160%로 상향 권장

2. **VoidRod 가격 장벽 극단적**: ElectricRod(225만) 대비 15배 가격  
   → 중간 단계 낚싯대 추가 또는 가격 조정 고려

3. **Unknown 물고기 확률 격차**: FishingRod(0.1%) vs VoidRod+풀버프(~3.5%) — 35배 차이  
   → 신규 플레이어 Unknown 도전 경험이 사실상 없음

### 🟡 보통

4. **슬라이더 속도 단조로움**: 모든 희귀도가 동일한 속도 범위(1.5~1.8s)  
   → Fantasy/Unknown은 1.0~1.3s 등 빠른 속도 추가 고려

5. **무게 분포 하향 편중**: 70.8%가 1~5kg 구간  
   → 게임 후반 플레이어에게 고무게 물고기 경험이 드물어 낮은 만족도

6. **날씨 쿨다운 비대칭**: Snow(×2.0 가치) 와 Rain(크기만 증가)의 가치 차이가 큼  
   → Rain에 소폭 코인 보너스(+20%) 추가 고려

### 🟢 경미

7. **버블 버프 가시성**: 버프 지속 시간과 스택 한도가 UI에 불명확  
8. **DrainRate와 히트존 불균형**: Unknown은 히트존 6~10%인데 드레인 9/s → 순수 운 의존성 높음

## 9. 스킬트리 — 실제 구현 현황

### 구조 개요

- **두 가지 재화**: 코인 (낚시로 획득) / Fishing Points (낚시 성공 1회당 +1, SkillPoints IntValue로 저장)
- **Root 노드** 선행 필수 — 해금 후 다른 스킬 구매 가능
- **선형 단계 구매** — 반드시 순서대로 (stage = currentLevel + 1 만 구매 가능)
- **노드 이름 규칙**: `SkillNode_스킬명_단계번호` (예: `SkillNode_LuckUp_2`)
- **GUI 위치**: `SkillTree > BackGround > Viewport > Canvas` 안에 ImageButton 배치

| PointBoost (풀업) | ~125M | ❗ 극단적 장기 목표 — FP 가속 수단 |

#### Fishing Points 스킬 투자 ROI

| 스킬 | 총 최대 투자 (FP) | 누적 Fishing Points 기준 |
|------|--------------|----------------------|
| FishingPoint | 5 FP | 5회 낚시 |
| WeightUp (풀업) | 800 FP | 약 800회 낚시 |
| PowerUp (풀업) | 300 FP | 약 300회 낚시 |
| HitZoneUp (풀업, 3단계) | 85 FP | 약 85회 낚시 |
| **전체 FP 풀업** | **~1,190 FP** | **약 1,190회 낚시** |

> PointBoost 풀업 시 낚시당 FP 4 획득 → 전체 FP 스킬 풀업에 필요한 낚시 횟수가 약 1/4로 감소.

#### 잠재적 밸런스 이슈

- **PointBoost Lv3 (1억 코인)**: 극단적으로 비쌈. ArcaneRod(6.5M) 대비 15배 이상. 초고수 전용 목표로 의도된 것으로 보임 — 문제 없음.
- **LuckUp + FriendBonus 조합**: 최대 LuckUp(+250%) + FriendBonus Lv3(+50% per friend). 친구 3명 온라인이면 +150% 추가 → 총 럭 +400%. ArcaneRod(+450%)와 비슷한 수준. **과도하지 않음**.
- **SellBonus Lv6 (+65%)**: 시간당 수익 1.65배. 게임 후반에 큰 영향 — 수익 곡선 가속화. 의도된 설계.
- **WeightUp**: 무게 구간 확률 이동 방식 → 높은 무게 확률 증가. 직관적이고 체감 명확.
- **HitZoneUp Lv3 (+15%)**: Studio 노드가 3단계까지만 존재. 서버는 5단계 정의되어 있으나 노드 미배치.

## 10. Fishing Points 경제 분석

### FP 획득 경로

| 방법 | 획득량 | 비고 |
|------|-------|-----|
| 낚시 성공 1회 | +1 FP | 기본 |

| PointBoost Lv2 | +3 FP | 낚시당 |

### FP 전체 스킬 풀업 소요 낚시 횟수

| PointBoost 단계 | 낚시당 FP | 전체 풀업(1,190 FP) 소요 |

| Lv0 (없음) | 1 | **1,190회** |
| Lv1 | 2 | **595회** |
| Lv2 | 3 | **397회** |
| Lv3 | 4 | **298회** |

> 오토피싱(AutoFish + AutoUp) 조합 시 분당 약 30~40회 낚시 → PointBoost 없이 약 30~40분이면 FP 스킬 전부 풀업 가능.
